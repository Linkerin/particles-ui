# Button

  githubLink="Button/Button.tsx"

The `Button` component allows users to trigger an action or event, such as
submitting a form, opening a dialog, canceling an action, or performing a delete
operation.

## Usage

### Basic usage

<ButtonDefault />

```jsx copy
import { Button } from '@snipshot/particles';

function MyComponent() {
  return <Button>Confirm</Button>;
}
```

### Variants

The `Button` component comes with five different variants: `filled`, `outlined`,
`text`, `soft`, and `elevated`.

<Tabs items={['Preview', 'Code']}>
  <Tabs.Tab><ButtonVariants /></Tabs.Tab>
  <Tabs.Tab>
    ```jsx copy
    import { Button, ButtonProps } from '@snipshot/particles';

    const variants: Array<ButtonProps['variant']> = [
      'filled',
      'outlined',
      'text',
      'soft',
      'elevated'
    ];

    function ButtonVariants() {
      return variants.map(variant => (
        <Button key={variant} variant={variant}>
          {variant}
        </Button>
      ));
    }
    ```

  </Tabs.Tab>
</Tabs>

### Colors

You can choose from one of the predefined color options by passing a `color`
props valiue: `primary`, `secondary`, `tertiary`, `info`, `success`, `warning`,
`error`.  
If you need an unstyled button, pass the `uncolored` value.

<Tabs items={['Preview', 'Code']}>
  <Tabs.Tab><ButtonColors /></Tabs.Tab>
  <Tabs.Tab>
    ```jsx copy
    import { Button, ButtonProps } from '@snipshot/particles';

    const colors: Array<ButtonProps['color']> = [
      'primary',
      'secondary',
      'tertiary',
      'success',
      'info',
      'warning',
      'error'
    ];

    function ButtonColors() {
      return colors.map(color => (
        <Button key={color} color={color}>
          {color}
        </Button>
      ));
    }
    ```

  </Tabs.Tab>
</Tabs>

### Radiuses

You can customize the border radius by providing a `radius` prop value.

<Tabs items={['Preview', 'Code']}>
  <Tabs.Tab><ButtonRadiuses /></Tabs.Tab>
  <Tabs.Tab>
    ```jsx copy

    const radiuses: Array<ButtonProps['radius']> = [
      'none',
      'xs',
      'sm',
      'md',
      'lg',
      'xl',
      'full',
      'round'
    ];

    function ButtonColors() {
      return radiuses.map(radius => (
        <Button key={radius} radius={radius} color="secondary">
          Radius: {radius}
        </Button>

    }
    ```

  </Tabs.Tab>

### Sizes

To make a button smaller or larger use the `size` props with the following
values: `xs`, `sm`, `md`, `lg`, `xl`.

  <Tabs.Tab>

    ```jsx copy
    const sizes: Array<ButtonProps['size']> = ['xs', 'sm', 'md', 'lg', 'xl'];

    function ButtonColors() {
      return sizes.map(size => (
        <Button key={size} size={size} color="tertiary">
          Size: {size}
        </Button>
    ```

### Loading state

The Button component supports a loading state with a loading spinner (or a
custom loading element) and optional loading text using `isLoading`,
`loadingText`, and `loadingElement` props.

<Tabs items={['Preview', 'Code']}>
<Tabs.Tab><ButtonLoading /></Tabs.Tab>
    <Tabs.Tab>
      ```jsx copy
      import { Button, RingSpinner } from '@snipshot/particles';

      function ButtonLoading() {
        return (
          <Button loadingText="Loading..." isLoading>
            Send
          </Button>
          <Button
            loadingElement={<RingSpinner size="xs" />}
            isLoading
            variant="elevated"
            color="tertiary"
          >
            Ring spinner
          </Button>
        );
      }
      ```
    </Tabs.Tab>

## Accessibility

The Button component follows
[WAI-ARIA](https://www.w3.org/WAI/ARIA/apg/patterns/button/) best practices to
ensure accessibility for all users. It includes semantic `<button>` element and
appropriate ARIA attributes to enhance compatibility with screen readers and
assistive technologies.

## Polymorphic component

`Button` is a polymorphic component. It means that using the `as` prop you can
pass any valid HTML element as a `string` or a React component and it will be
used a rendering base instead of a default `<button>` element.

<Tabs items={['Preview', 'Code']}>
<Tabs.Tab><ButtonPolymorphic /></Tabs.Tab>
    <Tabs.Tab>
      ```jsx copy
      import { Button } from '@snipshot/particles';

      function MyComponent() {
        return <Button as="a">I am a link</Button>;
      }
      ```
    </Tabs.Tab>

</Tabs>

## API Reference

| Prop           | Type                                                                                   | Default     | Description                                                                      |
| :------------- | :------------------------------------------------------------------------------------- | :---------- | :------------------------------------------------------------------------------- |
| as             | `React.ElementType`                                                                    | `button`    | HTML or React element that will be rendered by the component                     |
| color          | `'primary' \| 'secondary' \| 'tertiary' \|'success' \| 'info' \| 'warning' \| 'error'` | `'primary'` | Buttons's color scheme                                                           |
| dense          | `boolean`                                                                              | `false`     | Shrinks the button reducing paddings                                             |
| disabled       | `boolean`                                                                              | `false`     | Button's disabled state                                                          |
| isLoading      | `boolean`                                                                              | `false`     | Applies loading state styling and behaviour                                      |
| loadingElement | `React.ReactElement`                                                                   | `undefined` | Element that will be displayed in the loading state instead of a default spinner |
| loadingText    | `string`                                                                               | `''`        | Button's content that will be displayed in loading state                         |
| loadingSpinner | `boolean`                                                                              | `true`      | Disable a loading spinner in the loading state                                   |
| leftIcon       | `React.ReactElement`                                                                   | `undefined` | Icon element on the left                                                         |
| radius         | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl'`                                                 | `'md'`      | Border radius value                                                              |
| rightIcon      | `React.ReactElement`                                                                   | `undefined` | Icon element on the right                                                        |
| shadowOnPress  | `boolean`                                                                              | `false`     | Adds inset box-shadow imitating button press. Only for `filled` variant.         |
| shrinkOnPress  | `boolean`                                                                              | `false`     | Makes the button a bit smaller on press                                          |
| size           | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl'`                                                 | `'md'`      | Button's size value                                                              |
| variant        | `'filled' \| 'outlined' \| 'text' \| 'soft' \| 'elevated'`                             | 'filled'    | Styling variant                                                                  |
